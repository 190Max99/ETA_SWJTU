# About US
西南交通大学电子科技协会（始于1953）是一个致力于为西南交大学子提供一个学习和交流电子、嵌入式等相关知识平台的协会。在基于我们自身经验以及学习中遇到的困难，我们总结并为大家提供了入门电子与嵌入式等相关知识的适量引导，在大家追逐梦想的道路上尽一份力。

这个仓库有以下内容：
- 历届单片机大赛的优秀作品源码
- 嵌入式、电子等相关知识的入门教程
